An intelligent CRM platform built to bring AI-powered automation into everyday sales and support workflows.

- **Stack:** Next.js, FastAPI, GPT-4o, Whisper ,NLP,LLM,exotel, python,postgreSQL
- Combines a modern Next.js frontend with a FastAPI backend for fast, scalable performance

- Built for real client use cases in managing customer data and communication

### 📈 GrowthPilot AI
A marketing automation SaaS platform designed to simplify and scale campaign management.

- **Stack:** Next.js 15, Gemini API ,SQlite, python, nodejs
- Built on the latest Next.js 15 for performance and modern architecture
- Leverages the Gemini API to assist with campaign creation, content generation, and optimization
- Aimed at helping businesses automate repetitive marketing tasks and improve campaign efficiency

An Electron-based desktop tool for orchestrating multiple AI agents locally.

- **Stack:** Electron,vite,express,python,LangGrapg,croma DB,RAG,LLM,pytorch, Multi-Agent Architecture
- Designed to manage and coordinate multiple AI agents working in parallel
- Built as a local-first desktop application, giving users full control without relying on cloud infrastructure
- Focused on making complex multi-agent workflows more accessible and manageable
